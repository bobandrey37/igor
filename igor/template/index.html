Database access

Your raw database in XML can be found at /data/. You can append a path (technically: an XPath expression that resolves to a single node) to view a subsection of the database.

You can also run general XPath expressions by appending them to /evaluate/.

For testing purposes you can run internal commands by prefixing them with /internal/ (for example /internal/log), individual actions by prefixing with /action/ (for example /action/save) and plugins by prefixing with /plugin/.
